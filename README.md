# Qriza

A modern Flutter application for secure account creation and digital payment setup.  
Qriza enables users to register, link multiple payment methods (Bank, Easypaisa, JazzCash), and share payment details via QR code — all with a clean, intuitive UI.

Features
**Authentication**: Login & Signup with form validation
**Payment Setup**: Add Bank, Easypaisa, JazzCash
**QR Identity**: Generate and share encrypted payment QR codes
**Secure Data**: Encrypted storage for sensitive information
**Cross‑Platform**: Built with Flutter for Android & iOS
